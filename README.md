# 📰 Newsletter

Bem-vindo(a) ao repositório do projeto **Newsletter**! Esta é uma página web responsiva e moderna, focada na construção de um layout para captura de e-mails de subscritores.

Este projeto foi desenvolvido com o foco principal na aplicação do framework **Tailwind CSS**, permitindo uma estilização rápida, eficiente e focada no conceito *utility-first*.

## 💻 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica e organização da página.
- **Tailwind CSS:** Estilização através de classes utilitárias, garantindo um design moderno, flexível e responsivo sem a necessidade de escrever dezenas de linhas de CSS tradicional.

---

## ⚙️ Conceitos e Aprendizados Aplicados

- **Utility-First CSS:** Construção da interface visual aplicando classes pré-definidas do Tailwind diretamente nas tags HTML (ex: flexbox, espaçamentos, cores e tipografia).
- **Design Responsivo:** Utilização dos prefixos de *breakpoint* do Tailwind (como `md:`, `lg:`) para garantir que o formulário e o layout se adaptam perfeitamente a ecrãs de smartphones, tablets e computadores de secretária.
- **Produtividade no Front-end:** Redução significativa do tempo de desenvolvimento ao centralizar a estilização no próprio documento HTML.

---

## 🚀 Como executar o projeto localmente

Como o projeto é estático, a sua visualização local é muito simples:

1. Faça o clone deste repositório:
   > git clone https://github.com/gabrieldenti/newsletter.git

2. Navegue até à pasta do projeto:
   > cd newsletter

3. Abra o ficheiro `index.html` no seu navegador favorito. 
   *(Dica: Se utilizar o VS Code, recomendo a extensão **Live Server** para atualizar a página automaticamente a cada alteração guardada).*
